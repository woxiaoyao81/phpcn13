# phpcn13
记录自己在PHP中文网十三期在线培训学习时作业

在PHP中文网免费学习了一年多，也在这行工作一年多了，前端、后端都慢慢了解，但感觉开发项目时仍然存在诸多疑惑，于是下定决心逼下自己，报了十三期在线培训班，学习全栈开发。为了记录所学和所思，在这里按时间记录自己的作业和学习总结。
